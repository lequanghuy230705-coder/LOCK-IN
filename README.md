LOCK-IN SYSTEM
Hệ thống quản lý hành trình phát triển bản thân.

Công cụ sử dụng
Frontend: React, Tailwind CSS, Framer Motion, Axios.

Backend: Node.js, Express.

Database: Supabase.

Cách chạy dự án
1. Backend
Di chuyển vào thư mục backend, cài thư viện và chạy:
Bash
npm install express cors dotenv @supabase/supabase-js

Chạy sever.js
npm start

2. Frontend
Di chuyển vào thư mục frontend, cài thư viện và chạy:

Bash
npm install react-router-dom axios framer-motion lucide-react

chạy frontend
npm run dev
